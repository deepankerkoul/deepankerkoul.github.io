* **Northern Railway Network**
  * The project aimed at developing a well-established and running IPv4 network topology having redundant paths, terminal server configuration and centralized printing and ticketing management. The system comprises of data storage system and network cloud along with network load balancing and fail over clustering features along with high bandwidth and latency.  
  
  * Routing algorithm used is OSPF owing to its vendor neutrality and minimizing routing update traffic while allowing scalability as well as multi-vendor deployment. Routers and other network devices including PCs, Servers, Switches were configured individually using the Command Line Interface on IOS (Internetwork Operating System) – an OS platform developed for networking devices by Cisco.

  * DHCP was configured on individual networks of various zones and their respective departments in order to automate the scalability of network in future.

  * While working on the project was introduced to network simulation and setting up of an entire network (including router level configuration on the CLI) using the simulated environment provided by Cisco Packet Tracer.

  * Softwares/Technologies Used:

    * Cisco Packet Tracer (for design and simulation of the network)

* **IR Based Smart Lighting System**
  * IR based Smart Lighting System eliminates human error in everyday tasks such as switching off lights, by using an LDR circuit. However, it also takes into account special circumstances where light or other appliances may be needed even when input variables indicate otherwise, in such cases it receives IR input from the user.

  * It's a hardware based project that required designing the PCB and then fabricating the same in the lab followed by soldering the various components that execute the logic.

  * Softwares/Technologies Used:

    * Proteus Design (for design and pre-fabrication simulation of the circuit)
